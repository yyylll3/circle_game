# 画圆大作战 - 多人画圆评分排名

支持手机用手指在屏幕上画圆，自动评分（0-100），多人同时打开可各自绘制并查看实时排名。

## 运行方式

1. 进入目录：`cd circle-game`
2. 启动服务：`node server.js`
3. 浏览器打开：`http://localhost:3888`
4. 手机同网段访问：`http://你的电脑IP:3888`（如 `http://192.168.1.100:3888`）

## 发给朋友一起玩

**方式一：同一 WiFi（办公室 / 家里）**

1. 在你电脑上保持运行 `node server.js`。
2. 查本机 IP：在终端执行 `ipconfig`，找到「IPv4 地址」（如 `192.168.1.100`）。
3. 把链接发给朋友：**http://你的IP:3888**（例如 `http://192.168.1.100:3888`）。
4. 朋友用手机或电脑浏览器打开该链接即可，和你在同一排名里玩。

**方式二：朋友不在同一网络（用内网穿透）**

- 你电脑在自家/公司网络，朋友在外地时，需要用「内网穿透」把 3888 端口暴露到公网。
- 可选工具（选一个即可）：
  - **ngrok**：https://ngrok.com ，安装后执行 `ngrok http 3888`，会得到一个公网地址，把该地址发给朋友。
  - **cpolar**：https://www.cpolar.com ，国内服务，同样可把 3888 映射成公网链接发给朋友。
- 使用穿透时请勿在链接中暴露敏感信息，玩完可关掉穿透。

**方式三：部署到公网（推荐，大家随时可玩）**

把项目部署到一台公网可访问的服务器上，所有人打开同一个链接即可，无需内网穿透、不依赖你的电脑开机。

| 方式 | 特点 | 大致步骤 |
|------|------|----------|
| **Railway** | 国外，有免费额度，支持 Git 一键部署 | 注册 railway.app → 用 GitHub 导入本仓库 → 新建 Web Service，选 `circle-game`，启动命令填 `node server.js`，根目录选项目目录 → 部署后得到 `https://xxx.railway.app`，发给所有人即可。 |
| **Render** | 国外，有免费档 | 注册 render.com → New → Web Service → 连 GitHub 仓库，Runtime 选 Node，Build 填 `npm install`（可选，本项目无依赖），Start 填 `node server.js` → 部署后得到公网链接。 |
| **国内云服务器** | 阿里云/腾讯云/华为云 轻量应用服务器 | 买一台轻量机（约几十元/年），系统选 Linux → SSH 登录后安装 Node.js，把 `circle-game` 上传或 git clone → 在项目目录执行 `node server.js`（或用 pm2 常驻），在安全组/防火墙放行 3888 端口 → 用服务器公网 IP:3888 访问。若想用 80 端口或域名，可再配 Nginx 反向代理。 |

部署后排名数据存在该服务器内存里；服务器重启会清空排名，需要持久化可后续改造成文件或数据库存储。

## 玩法

- 在画布上用手指/鼠标画一个圆，松手后自动给出得分（越圆分数越高）。
- 输入昵称后点击「提交分数」参与排名。
- 排名每 8 秒自动刷新，多人可同时玩并看到同一份排名。

## 技术说明

- 前端：HTML5 Canvas，支持触摸与鼠标绘制。
- 评分：根据轨迹与拟合圆的偏差、首尾闭合程度计算 0-100 分。
- 后端：Node.js 内置 HTTP 服务，内存存储排名（重启后清空）。
