# 画圆大作战 - 发布给朋友访问（联网）

项目已用 Supabase 做全网排名，前端是静态页面，用 **GitHub Pages** 即可免费发布，朋友打开链接即可联网玩。

---

## 一、用 GitHub Pages 发布（推荐）

### 1. 确保代码已推到 GitHub

- 仓库里要有 `index.html`、`supabase-setup.sql` 等（可不需要 `server.js`，因为排名走 Supabase）。

### 2. 开启 GitHub Pages

1. 打开你的仓库页面，点击 **Settings（设置）**。
2. 左侧找到 **Pages**。
3. 在 **Build and deployment** 里：
   - **Source** 选 **Deploy from a branch**。
   - **Branch** 选 `main`（或你放代码的分支），右边选 **/ (root)**。
4. 点 **Save**。
5. 等 1～2 分钟，刷新 Pages 设置页，顶部会显示：
   - **Your site is live at `https://<用户名>.github.io/<仓库名>/`**

### 3. 把链接发给朋友

- 把这个地址发给朋友，例如：  
  `https://你的用户名.github.io/circle-game/`
- 朋友用浏览器打开即可联网玩，提交的分数会进 Supabase 全网排名。

---

## 二、可选：自定义域名

- 在 Pages 设置里可以填 **Custom domain**（例如 `game.yourdomain.com`），按提示在域名服务商那里加一条 CNAME 解析即可。

---

## 三、注意事项

| 项目 | 说明 |
|------|------|
| **无需开 server** | 排名用 Supabase，前端直接连 Supabase，不依赖 `node server.js`。 |
| **HTTPS** | GitHub Pages 自带 HTTPS，地址是 `https://...`。 |
| **Supabase 配置** | `index.html` 里已填的 `SUPABASE_URL` 和 `SUPABASE_ANON_KEY` 会随页面一起发布，排名照常可用。 |
| **清空排名** | 只有知道重置密码的人可以点「清空排名」，密码在 Supabase 的 `app_config` 里。 |

## 四、在 Vercel 上发布

### 1. 登录 Vercel

- 打开 **[vercel.com](https://vercel.com)**，用 **GitHub** 账号登录（或 GitLab / Bitbucket）。
- 授权 Vercel 访问你的 GitHub 仓库。

### 2. 导入项目

1. 点击 **Add New…** → **Project**（或首页 **Import Project**）。
2. 在列表里找到 **circle-game**（或你的仓库名），点 **Import**。
3. **Configure Project** 页面一般不用改：
   - **Framework Preset**：选 **Other** 或 **Vite** 无所谓，本项目是纯静态。
   - **Root Directory**：保持默认（`.`）。
   - **Build Command**：可留空（没有构建步骤）。
   - **Output Directory**：留空或填 `.`（根目录就是网站根目录）。
4. 点击 **Deploy**。

### 3. 等待部署

- 大约 30 秒～1 分钟，部署完成。
- 会显示 **Congratulations!** 和一个网址，形如：  
  `https://circle-game-xxx.vercel.app`  
  或你自定义的子域名。

### 4. 访问与分享

- 点 **Visit** 或直接打开该链接，即可玩游戏。
- 把这个链接发给朋友，他们联网打开即可，排名照常使用 Supabase。

### 5. 之后更新代码

- 只要往 GitHub 仓库 **push** 新代码，Vercel 会自动重新部署，无需再点一次 Deploy。

---

## 五、其他发布方式（可选）

- **Netlify**：在 [netlify.com](https://netlify.com) 用 GitHub 导入项目，得到 `xxx.netlify.app` 网址，步骤类似 Vercel。

任选其一即可；发布后把页面链接发给朋友即可联网使用。
